MCP endpoint (Streamable HTTP), in-cluster:
  http://{{ include "mcs.fullname" . }}.{{ .Release.Namespace }}.svc:{{ .Values.service.port }}/mcp

Try it locally:
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "mcs.fullname" . }} 3000:{{ .Values.service.port }}
  curl -s localhost:3000/healthz

Connect an agent (Claude Code):
  claude mcp add --transport http context-stream http://localhost:3000/mcp \
    --header "Authorization: Bearer <token>" --header "X-Agent-Name: my-agent"

Ingest events from CI/webhooks:
  curl -X POST http://localhost:3000/ingest/deployments \
    -H "Authorization: Bearer <token>" -H "content-type: application/json" \
    -d '{"type":"ci.build.failed","payload":{"repo":"api"}}'
